# Simple Forum

A basic web forum built with Node.js, Express, MongoDB, and Tailwind CSS.

## Features
- Create threads
- Reply to threads
- Delete threads
- No login required

## Setup

1. Install dependencies:
```
npm install
```

2. Make sure MongoDB is running on your computer

3. Start the server:
```
npm start
```

4. Open browser and go to: http://localhost:3000

## Requirements
- Node.js
- MongoDB
