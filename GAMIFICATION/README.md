# 🎯 Animated Quiz App

Beautiful quiz application with 25 questions featuring smooth animations, Lottie badges, and confetti celebrations.

## ✨ Features

- **25 Quiz Questions** (8 Simple, 10 Moderate, 12 Hard)
- **Framer Motion** - Smooth scale, blur, and popup animations
- **Lottie Animations** - Beautiful badge animations for results
- **Confetti Effects** - Celebration animations using canvas-confetti
- **API-Driven** - Simulated API responses trigger animations
- **Responsive Design** - Works on all devices
- **Progress Tracking** - Visual progress bar and score tracking

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## 🎨 Technologies

- React 18
- Framer Motion (animations)
- Lottie React (badge animations)
- Canvas Confetti (celebration effects)
- Vite (build tool)

## 📱 Features Breakdown

### Animations
- Scale & rotate transitions on question cards
- Blur backdrop on result popups
- Spring physics for natural motion
- Staggered option animations

### Quiz Structure
- Simple: Questions 1-8 (basics)
- Moderate: Questions 9-18 (intermediate)
- Hard: Questions 19-30 (advanced)

### Visual Feedback
- Color-coded difficulty badges
- Animated progress bar
- Success/failure popups with Lottie
- Confetti on correct answers
- Trophy animation on completion

Enjoy the quiz! 🎉
